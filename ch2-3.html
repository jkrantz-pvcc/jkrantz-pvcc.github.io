Chapter 2: Briefly define:

  1. natural language is the language we speak and write

  3. sequential operations execute code in a straight line top to bottom then stops
  4. input operations give the computing agent data from the outside world to be used
  5. output operations send results from the computing agent to the outside world
  6. conditional statements are the question asking operations of an algorithm(if/then/else)
  7. boolean condition are true/false yes/no
